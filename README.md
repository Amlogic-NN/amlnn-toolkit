# amlnn-toolkit
Amlogic NNDDK (development package, documentation, and examples). Includes front-end tools for model import, model quantization, and model conversion, as well as deployment-related development packages such as nnsdk and llmsdk.

Development platform support:
  PC: x86-ubuntu/debian, for offline model conversion.Supported operating systems: ubuntu, debian, yocto, Android, and freertos.
  All based on an arm64-ubuntu environment for model conversion and deployment.
